# homebrew-neokikoeru

This is the official [Homebrew](https://brew.sh/) tap for [Neokikoeru](https://github.com/vscodev/neokikoeru). You can use it to install Neokikoeru on MacOS and Linux using Homebrew.

## Install

To install Neokikoeru with Homebrew, aka. `brew`, run the following commands:

```bash
brew tap vscodev/neokikoeru
brew install neokikoeru
```

## Upgrade

To upgrade, it is recommended that you run the `brew upgrade` command.

```bash
brew upgrade neokikoeru
```

## Uninstall

If you want to uninstall Neokikoeru, run the following commands:

```bash
brew uninstall neokikoeru
brew untap vscodev/neokikoeru
```
